
# 10 Audit Documentation

## Projektziel

In diesem Modul wurde eine Audit-Dokumentation für das Enterprise IT Administrator Lab erstellt. Ziel war es, die umgesetzten technischen Kontrollen, Risiken, Nachweise und Ergebnisse strukturiert zu dokumentieren.

Dieses Modul verbindet technische IT-Administration mit Governance, Risk & Compliance (GRC).

## Umgebung

- Windows Server 2022
- Active Directory Domain Services
- File Server
- Group Policy Management
- Windows Server Backup
- pfSense Firewall
- Microsoft 365 / Microsoft Entra ID
- Monitoring und Logs
- GitHub-Dokumentation
