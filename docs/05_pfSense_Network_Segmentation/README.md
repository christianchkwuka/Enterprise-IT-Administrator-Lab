
# pfSense Network Segmentation

## Objective
This module demonstrates network segmentation using pfSense firewall rules. The goal is to separate HR, Finance, IT, and DMZ networks and control communication between them.

## Implemented Controls
- Interface assignment
- VLAN or virtual network segmentation
- DHCP configuration
- Inter-network firewall rules
- Blocked traffic validation
- Allowed traffic validation
- Firewall log review

## Security Purpose
Network segmentation reduces lateral movement, protects sensitive systems, and enforces least privilege communication between departments.
